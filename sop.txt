2. Project Vision
SMS wants a one-stop community platform for Muslims in Suffolk — all the information they need in one place, regularly updated, and accessible to the whole community. The site should feel clean, professional, and trustworthy, in the style of the East London Mosque website (eastlondonmosque.org.uk).

Design references provided by client:
East London Mosque — eastlondonmosque.org.uk (primary reference)

5. Prayer Times Integration
The timetables page will display prayer times for all 4 masjids in Suffolk. Data sources provided by client:

Nawra Mosque — nawracymosque.org/ipswich-prayer-times/
Masjid Taqwa — prayersconnect.com/mosques/82611734-masjid-taqwa-suffolk-england-united-kingdom
Ipswich Mosque — ipswichmosque.org/prayer_time.php
Shah Jalal Masjid — shahjalalmasjidipswich.co.uk/

Approach: Timetable PDFs/images uploaded via admin dashboard and stored on Cloudflare R2. Public page fetches latest record from DB and renders. Admin can update weekly without any technical knowledge.

6. Admin Dashboard
A protected route at /admin, secured with a password-based httpOnly cookie (no third-party auth library required). The dashboard will include separate management sections for each content type:

/admin/timetable — upload PDF or image per masjid
/admin/events — create, edit, delete events
/admin/classes — manage class schedule
/admin/announcements — TipTap rich text editor

No technical knowledge required from the committee — all content management is through simple forms and file uploads.
